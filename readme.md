

# Teensy 48

![PXL_20240110_124119080](https://github.com/opelpanfan/OPF_ECU/assets/12942077/a79a7790-c60c-4dd3-baf1-0e0608057663)

## Features

- 8 Injectors
- 8 Coils
- 8 Low Side Switch
- 2 VR / 2 Hall Triggers
- 4 Digital Inputs
- 8 Analogue 0-5v inputs
- 1 Wideband (14point7)
- 1 Canbus
- 4bar map
- Onboard Barometer
